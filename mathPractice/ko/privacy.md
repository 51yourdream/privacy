# 📋 개인정보처리방침 템플릿 - 수학 학습 도우미 앱

## 개인정보처리방침

**시행일**: 2025년 6월 6일
**최종 업데이트**: 2025년 6월 6일

### 1. 소개

"수학 학습 도우미" 애플리케이션(이하 "저희 앱" 또는 "본 앱")을 선택해 주셔서 감사합니다. 본 개인정보처리방침은 저희가 귀하의 정보를 어떻게 수집, 사용, 저장, 보호하는지에 대해 설명합니다.

저희는 사용자 개인정보 보호의 중요성을 깊이 이해하고 있으며, 관련 개인정보보호 법규를 엄격히 준수합니다.

### 2. 정보 수집

#### 2.1 저희가 수집하는 정보의 유형

**학습 데이터**:
- 연습 성과 및 답안 기록
- 학습 진도 및 통계 정보
- 앱 사용 시간
- 오답 기록 및 복습 상황

**기기 정보**:
- 기기 모델 및 운영체제 버전
- 앱 버전 정보
- 기기 고유 식별자(구독 관리용)

**구독 정보**:
- 구독 상태 및 유효 기간
- 구매 기록(Apple App Store를 통해 처리)

#### 2.2 저희가 수집하지 않는 정보

저희는 다음 정보를 수집하지 않을 것을 약속합니다:
- 개인 식별 정보(이름, 주소, 전화번호 등)
- 사진, 동영상 또는 음성 기록
- 위치 정보
- 연락처 또는 연락처 정보
- 소셜 미디어 계정 정보

### 3. 정보 사용 방식

저희는 수집한 정보를 다음 목적으로 사용합니다:

**서비스 제공**:
- 개인화된 학습 경험
- 학습 진도 추적
- 학습 보고서 및 통계 분석 생성

**앱 개선**:
- 앱 성능 최적화
- 버그 및 기술적 문제 수정
- 새로운 기능 개발

**구독 관리**:
- 구독 상태 확인
- 구매 및 환불 처리

### 4. 정보 저장 및 보안

#### 4.1 로컬 저장 우선

모든 학습 데이터와 개인 설정은 귀하의 기기에 로컬로 저장되며, 저희 서버에 업로드되지 않습니다. 이를 통해 다음이 보장됩니다:
- 완전한 데이터 제어권
- 오프라인 기능 이용 가능성
- 최대한의 개인정보 보호

#### 4.2 데이터 보안 조치

저희는 귀하의 정보를 보호하기 위해 다음 조치를 취합니다:
- 데이터 암호화 저장
- 안전한 데이터 전송 프로토콜
- 정기적인 보안 검토
- 접근 권한 제어

#### 4.3 데이터 보존

- 학습 데이터: 앱을 삭제할 때까지 기기에 저장
- 구독 정보: Apple App Store 정책에 따라 보존
- 임시 데이터: 불필요한 캐시 데이터를 정기적으로 정리

### 5. 제3자 서비스

저희는 다음 제3자 서비스를 사용합니다:

#### 5.1 RevenueCat(구독 관리)
- 용도: 앱 내 구매 및 구독 관리
- 수집 정보: 기기 식별자, 구매 기록
- 개인정보처리방침: https://www.revenuecat.com/privacy

#### 5.2 Apple App Store
- 용도: 앱 배포 및 결제 처리
- 수집 정보: 구매 정보, 기기 정보
- 개인정보처리방침: https://www.apple.com/privacy/

### 6. 사용자 개인정보 보호

#### 6.1 데이터 보호
저희는 개인정보보호 규정을 엄격히 준수합니다:

**사용자 정보 보호**:
- 개인 식별 정보 미수집
- 모든 데이터의 로컬 저장
- 제3자 광고 미포함
- 소셜 기능 미포함
- 사용자의 완전한 앱 사용 제어

#### 6.2 사용자 권리
사용자는 다음 권리를 누립니다:
- 자신의 학습 데이터 조회
- 모든 학습 기록 삭제
- 앱 기능 및 설정 제어
- 언제든지 앱 사용 중단

#### 6.3 사용 동의
본 앱 사용은 사용자가 다음에 동의함을 의미합니다:
- 본 개인정보처리방침 조항
- 감독하에 앱 사용(해당하는 경우)
- 데이터 수집 및 사용 방식 이해

### 7. 국제 사용자

저희 앱은 전 세계 여러 국가와 지역에서 서비스를 제공합니다. 서로 다른 지역의 사용자에 대해 저희는 다음을 포함하되 이에 국한되지 않는 현지 개인정보보호 법규를 준수합니다:

- 유럽연합 일반 데이터 보호 규정(GDPR)
- 캐나다 개인정보보호 및 전자문서법(PIPEDA)
- 호주 개인정보보호법

### 8. 귀하의 권리

적용 가능한 개인정보보호법에 따라 귀하는 다음 권리를 누립니다:

**접근권**: 저희가 수집한 정보 조회
**정정권**: 부정확한 정보 수정
**삭제권**: 귀하의 정보 삭제 요구
**제한권**: 정보 처리 제한
**데이터 이동권**: 귀하의 데이터 사본 획득

### 9. 개인정보처리방침 업데이트

저희는 본 개인정보처리방침을 정기적으로 업데이트할 수 있습니다. 중요한 변경사항은 다음 방법으로 알려드립니다:
- 앱 내 알림
- 앱 설명 업데이트
- 저희 웹사이트에 게시

귀하가 앱을 계속 사용하는 것은 업데이트된 개인정보처리방침에 동의함을 의미합니다.

### 11. 데이터 삭제 안내

**앱 데이터 삭제 방법**:
1. 앱 삭제: 모든 로컬 데이터가 영구 삭제됩니다

**삭제 후 영향**:
- 모든 학습 기록이 손실됩니다
- 사용자 정의 설정이 기본값으로 복원됩니다
- 구독 상태는 영향을 받지 않습니다(App Store에서 관리 필요)

### 12. 데이터 유출 대응

저희는 로컬 저장 우선 전략을 채택하고 있지만, 데이터 유출 사건이 발생할 경우 저희는 다음을 수행합니다:
- 즉시 조사 및 처리
- 72시간 이내에 관련 규제 당국에 신고
- 영향받은 사용자에게 신속한 통지
- 추가 유출 방지를 위한 조치 실시

### 13. 국경 간 데이터 전송

구독 확인에 필요한 최소한의 데이터를 제외하고는 국경 간 데이터 전송을 하지 않습니다. 필요한 데이터 전송은 모두 다음을 수행합니다:
- 적절한 보안 조치 채택
- 대상 국가의 법적 요구사항 준수
- 필요한 법적 근거 확보
